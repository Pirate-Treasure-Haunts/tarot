# TAROT
Alchemical Tarot of the Human Condition

A modern, symbolic tarot system and companion software exploring inner states,
decision-making, and narrative meaning through a deliberately human-centred lens.

This repository contains the **core tarot deck data**, **card logic**, and
**cross-platform tooling** used to generate readings and explore the structure
of the deck.

The project is intentionally:
- reflective rather than predictive
- symbolic rather than mystical
- exploratory rather than prescriptive

No claims of divination, prophecy, or supernatural authority are made.

---

## ‚ú® What This Is

- A **106-card tarot deck**, including:
  - Traditional Major Arcana
  - Traditional Minor Arcana
  - A bespoke extended Major Arcana set
- A **software tool** that can:
  - draw cards programmatically
  - generate structured readings
  - support experimentation, remixing, and extension
- A **creative and reflective framework**, designed to be used for:
  - journaling
  - storytelling
  - personal reflection
  - symbolic analysis
  - artistic or narrative projects
