# Changelog
All notable changes to **Alchemical Tarot of the Human Condition** will be documented in this file.

This project follows a pragmatic versioning approach:
major versions reflect structural or conceptual shifts,
minor versions reflect feature growth or stabilisation,
patch versions reflect fixes and packaging improvements.

---

## [4.0.1] – 2026-01-XX
### Added
- Fully automated cross-platform build pipeline
- Windows installer and portable ZIP distribution
- Linux `.deb` and `.rpm` packages
- macOS distributable ZIP build
- End-user distribution documentation (`README_DIST.txt`)
- Project governance documents:
  - `SECURITY.md`
  - `CONTRIBUTING.md`
  - `CODE_OF_CONDUCT.md`
- Clear personal-use licensing with explicit permission for:
  - modding
  - remixing
  - screenshots and streaming

### Changed
- Project positioned as a reflective symbolic system rather than predictive software
- Distribution strategy clarified and documented
- Build artefacts standardised across platforms

### Fixed
- CI instability across operating systems
- Packaging inconsistencies between Linux distributions
- Windows installer payload resolution issues

---

## [4.0.0] – 2026-01-XX
### Added
- Complete **100-card tarot deck**, including:
  - Traditional Major Arcana
  - Traditional Minor Arcana
  - Extended custom Major Arcana set
- Programmatic card draw engine
- Deterministic deck handling and shuffling
- Structured card metadata and meaning schema
- Initial command-line interface for drawing cards

### Changed
- Project architecture stabilised for public release
- Deck definitions separated cleanly from application logic

### Notes
This release represents the first *complete* expression of the deck as a finished symbolic system.

---

## [3.x] – Development Cycle
### Added
- Iterative development of extended Major Arcana concepts
- Experimental card meaning structures
- Early deck validation and balance testing

### Changed
- Multiple revisions to card naming, ordering, and thematic grouping
- Gradual shift away from mystical framing toward human-centred symbolism
