# Contributing to Alchemical Tarot of the Human Condition

Thank you for your interest in contributing.

This project values:
- clarity
- respect
- intentional design
- thoughtful change over rapid churn

---

## Ways to Contribute

- Bug reports
- Documentation improvements
- Deck extensions or variants
- Tooling improvements
- Thoughtful discussion and critique

---

## Code Style

- Keep changes minimal and intentional
- Avoid unnecessary dependencies
- Prefer readability over cleverness
- Document non-obvious decisions

---

## Workflow
